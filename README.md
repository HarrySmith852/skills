# Skills

Personal agent skills for use with the `skills` CLI and skills.sh.

## Available Skills

- `supower-power`: choose and enforce the right Superpowers-style workflow for development tasks.

## Install

```powershell
npx skills add HarrySmith852/skills --skill supower-power
```

This repository is intended to appear on skills.sh after it is installed through the `skills` CLI.
