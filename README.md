# Skills

Personal agent skills for use with the `skills` CLI and skills.sh.

## Available Skills

- `supower-power`: choose and enforce the right Superpowers-style workflow for development tasks.

## Install

```powershell
npx skills add farooq/skills --skill supower-power
```

Replace `farooq/skills` with the GitHub owner and repository name if this repo is published under a different account.
